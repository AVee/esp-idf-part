# esp-idf-part

A library for parsing and generating ESP-IDF partition tables. Supports parsing from and generating to both CSV and Binary files.

## Resources

- [ESP-IDF Partition Table API Guide](https://docs.espressif.com/projects/esp-idf/en/latest/esp32/api-guides/partition-tables.html)
